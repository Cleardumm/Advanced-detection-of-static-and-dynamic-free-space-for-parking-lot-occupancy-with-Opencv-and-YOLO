If you want to use segmentation to divide dynamic parking spaces, run main_byseg.py
If you want to use YOLO to divide dynamic parking spaces, run main_byyolo.py
The output of the current code is a log file, name is output.txt
If you want to output video, please uncomment the lines 283, 284, and 285 of main_byseg.py and lines 291, 292, 293 of main_byyolo.py, the output video is in the file RAW_data\demo\demo_video
